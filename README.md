🧩 Nuevo reto de lógica
El reto de la suma oculta  
Escribe un programa que:

Genere un arreglo de 10 números aleatorios entre 1 y 100.

Oculte uno de esos números y muestre los demás.

El jugador debe adivinar cuál es el número oculto.

El programa da pistas: si el número ingresado es mayor o menor que el oculto.

Al acertar, muestra cuántos intentos tomó.
